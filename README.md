## 🚀 Sobre o Projeto

Este é um portfólio pessoal ultra-tecnológico desenvolvido por **Reinaldo Passos Jr**, um desenvolvedor frontend apaixonado por tecnologia e design. O site apresenta um design cyberpunk com animações avançadas, efeitos visuais neon e uma experiência interativa única.

### ✨ Características Principais

- 🎨 **Design Cyberpunk**: Cores neon (ciano, magenta, verde) em fundo escuro
- 🌍 **Multi-idioma**: Suporte completo para português e inglês
- ⚡ **Animações Avançadas**: Particles, typing effect, scroll reveal, glitch effects
- 📱 **Responsivo**: Design mobile-first que funciona perfeitamente em todos os dispositivos
- 🔲 **Glassmorphism**: Efeitos de vidro translúcido modernos
- 🎯 **Performance**: Código otimizado e carregamento rápido

## 🌐 Sistema de Idiomas

O site suporta dois idiomas: **Português** e **Inglês**. O idioma padrão é português, mas você pode alternar facilmente clicando nas bandeiras no topo direito da página.

### Como Usar:
1. **Clique** na bandeira 🇧🇷 para português ou 🇬🇧 para inglês
2. **Navegue** entre as páginas - o idioma permanece salvo
3. **Typing effect** muda automaticamente conforme o idioma

## 🔧 Personalização

### Alterar Cores
Edite as variáveis CSS em `css/style.css`:

```css
:root {
    --neon-cyan: #00f3ff;    /* Mude para sua cor preferida */
    --neon-magenta: #ff00ff; /* Mude para sua cor preferida */
}
```

### Adicionar Novo Idioma
1. Adicione traduções em `js/translations.js`
2. Crie novas páginas com sufixo (ex: `index-es.html`)
3. Adicione nova bandeira em `img/flags/`

### Modificar Animações
Edite `js/script.js` para personalizar:
- Velocidade das particles
- Efeitos de digitação
- Animações de scroll
