## 🚀 About the Project

This is an ultra-technological personal portfolio developed by **Reinaldo Passos Jr**, a passionate frontend developer who loves technology and design. The site features a cyberpunk design with advanced animations, neon visual effects, and a unique interactive experience.

### ✨ Key Features

- 🎨 **Cyberpunk Design**: Neon colors (cyan, magenta, green) on dark background
- 🌍 **Multi-language**: Full support for Portuguese and English
- ⚡ **Advanced Animations**: Particles, typing effect, scroll reveal, glitch effects
- 📱 **Responsive**: Mobile-first design that works perfectly on all devices
- 🔲 **Glassmorphism**: Modern translucent glass effects
- 🎯 **Performance**: Optimized code and fast loading

## 🌐 Language System

The site supports two languages: **Portuguese** and **English**. The default language is Portuguese, but you can easily switch by clicking the flags at the top right of the page.

### How to Use:
1. **Click** on the 🇧🇷 flag for Portuguese or 🇬🇧 flag for English
2. **Navigate** between pages - the language remains saved
3. **Typing effect** automatically changes according to the language

## 🚀 How to Run

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for Font Awesome)

### Steps to View
1. **Clone** or **download** the repository
2. **Open** any `.html` file in the browser
3. **Alternatively**, use a local server:
   ```bash
   # With Python 3
   python -m http.server 8000

   # With Node.js
   npx serve .

   # With PHP
   php -S localhost:8000
   ```

- **Portuguese**: `index.html`

## 🔧 Customization

### Changing Colors
Edit the CSS variables in `css/style.css`:

```css
:root {
    --neon-cyan: #00f3ff;    /* Change to your preferred color */
    --neon-magenta: #ff00ff; /* Change to your preferred color */
}
```

### Adding a New Language
1. Add translations in `js/translations.js`
2. Create new pages with suffix (e.g., `index-es.html`)
3. Add new flag in `img/flags/`

### Modifying Animations
Edit `js/script.js` to customize:
- Particle speed
- Typing effects
- Scroll animations
